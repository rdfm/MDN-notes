## Creating a navigation menu

**Project Pages** :

- index.html
- projects.html
- pictures.html
- social.html

```html
In index.html page:
<ul>
  <li>Home</li>
  <li><a href="pictures.html">Pictures</a></li>
      <li><a href="projects.html">Projects</a></li>
      <li><a href="social.html">Social</a></li>
    </ul>
```
